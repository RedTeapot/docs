## Open-source license

The open-source edition is published under the **MIT License** — one of the most permissive licenses available. You can use it in personal projects, open-source software, and commercial applications without paying a fee or requesting permission.

<Card title="MIT License — full text" icon="file-text" href="https://opensource.org/licenses/MIT">
  Read the complete MIT License. It permits use, copy, modification, merging, publishing, distributing, sublicensing, and selling with minimal restrictions.
</Card>

## Compatibility

Before choosing a plan, confirm your environment meets the following requirements. All three editions share the same compatibility baseline.

### Framework

| Requirement | Supported |
|---|---|
| Vue 2.x | ✅ |
| Vue 3.x | ❌ (not supported) |

### Browsers

| Browser | Minimum version |
|---|---|
| Chrome | 60+ |
| Firefox | 55+ |
| Safari | 12+ |
| Edge (Chromium) | 79+ |
| Internet Explorer | ❌ (not supported) |

### Build tools

| Tool | Minimum version |
|---|---|
| Webpack | 4+ |
| Vite | 3+ |
| Rollup | 2+ |

<Note>
  XcaTree also ships a UMD bundle (`xca-tree.umd.js`) for use without a build tool — drop it in a `<script>` tag and it registers itself as a global Vue component.
</Note>
